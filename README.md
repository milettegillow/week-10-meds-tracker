# ADHD Meds Tracker 💊

**Week 10 of my [2026 weekly build series](https://www.milettegillow.com)**

A physical LED tracker built with an Arduino Uno to stop me forgetting (or double-dosing) my ADHD meds.

## How it works

- 🟡 **Yellow** — meds not taken yet
- 🟢 **Green** — taken! Click the button once when you take them
- 🔴 **Red** — taken twice, slow down!
- Automatically resets to yellow 12 hours after going green
- Hold the button for 2 seconds to manually reset at any time

## Built with

- Arduino Uno
- Kitronik Inventor's Kit (RGB LED, push button, resistors, breadboard)
- A Jo Malone box and some googly eyes
